A single-page quick reference for **Neovim (LazyVim)**, **tmux**, **WezTerm**, and
**Git / GitHub** (`gh`, `lazygit`, `octo.nvim`) — styled in Catppuccin Mocha, with a
print/PDF layout.

**Live site:** https://syntheticnerd.github.io/terminal-cheatsheet/

Built as a self-contained `index.html` (no build step, no dependencies).
