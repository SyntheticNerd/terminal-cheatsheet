Neovim

LazyVim distribution · leader = Space
Docs LazyVim Keymaps ↗ LazyVim Config ↗ Neovim Manual ↗
The mental model: Neovim is modal. You're almost always in Normal mode (for moving & commands), not typing text. Press i to enter Insert mode and type; press Esc to go back. If a keypress does something weird, you're probably in the wrong mode — hit Esc.

tmux

terminal multiplexer · prefix = Ctrl a
Docs Getting Started ↗ Manual — all keys & options ↗ Wiki ↗
The mental model: a session holds windows (like tabs), and each window holds panes (splits). Everything survives disconnection — detach and the programs keep running. Every shortcut below is: press Ctrl a, release, then the key.

Sessions (typed in shell)

tmux
Start a new session
tmux new -s dev
New named session ("dev")
tmux ls
List running sessions
tmux a
Attach to last session
tmux a -t dev
Attach to a named session

WezTerm

the terminal · opens straight into WSL Ubuntu
Docs Key Bindings ↗ Default Keys ↗ Config Files ↗
tmux vs WezTerm splits: WezTerm has its own tabs and panes, but for real work prefer tmux inside it — tmux survives detach, closed windows, and SSH. Use WezTerm tabs/panes for quick, throwaway layouts.

Where your configs live

Neovim (LazyVim)

~/.config/nvim/

Theme set in lua/plugins/colorscheme.lua. Add plugins as files under lua/plugins/. Restart nvim to apply.

tmux

~/.tmux.conf

Reload without restarting: Ctrl a then r. Or tmux source ~/.tmux.conf.

WezTerm

C:\Users\<you>\.config\wezterm\wezterm.lua

Lives on the Windows side. Auto-reloads the instant you save — no restart needed.
